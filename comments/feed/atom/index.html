]]> Comment on Selectivizr with CSS on a sub-domain by James James /?p=886#comment-2608 2012-05-16T11:22:35Z 2012-05-16T11:22:35Z One solution is to use javascript to rewrite the stylesheet domains to be relative to the page.

This can be added in the after the stylesheet have been loaded (amend regex as required)

<!--[if (gte IE 6)&(lte IE 8)]>-->

$('link[rel=stylesheet]').each(function() {
var new_href = $(this).attr('href').replace(/http:\/\/.*cdn\..*\.com\//,'/');
$(this).attr('href', new_href);
});
